# Bulk Screenshots

A simple Node.js program to take screenshots of multiple URLs using [the ScreenshotOne API](https://screenshotone.com/).

Check out more examples in the [ScreenshotOne examples](https://github.com/screenshotone/examples) repository.

Check out our tips on [how to take screenshots of multiple URLs with the ScreenshotOne API](https://screenshotone.com/docs/guides/bulk-screenshots/).

## Prerequisites

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
